🌊 Buoyancy & Wave Simulation in GoGo言語（Golang）と2Dゲームエンジン「Ebitengine」を使用して作成された、波の要素と物体の形状（球体）を考慮したインタラクティブな浮力物理シミュレーションです。
空中から水面に落下した球体が、流体抵抗やアルキメデスの原理（球欠計算）に基づき、波の傾き（斜面）によって左右にぷかぷかと揺さぶられながら流されていくリアルな挙動をGUIで可視化します。

🚀 特徴本格的な浮力計算（球欠対応）: 球体が水に沈む深さに応じて、非線形に変化する浸水体積（円欠の断面積）を毎フレーム厳密に計算。
動的な波（サイン波）の表現: 時間経過とともに変化する水面をシミュレート。波力（水平方向の移動）の再現: 水面の傾き（微分係数）から法線ベクトルを割り出し、波の斜面によって物体が横に押し出される挙動を再現。

📊 物理ロジックの概要1. 浮力（アルキメデスの原理）物体が押しのけた流体の重さ（体積 $\times$ 密度 $\times$ 重力加速度）に比例した浮力が上向きに働きます。
本シミュレーションでは2D表現として、球体が水面と交差した部分の「断面積」を以下の数式（円欠の面積公式）で動的に計算しています。
$$\text{Area} = r^2 \arccos\left(\frac{r-h}{r}\right) - (r-h)\sqrt{2rh - h^2}$$( $r$: 球の半径, $h$: 沈んでいる深さ )
2. 波の傾きと横流力の発生水面を $y = A \sin(fx - st)$ の関数で定義し、その微分係数（傾き）から物体の受ける浮力を法線方向に分散させています。
これにより、斜面を滑り落ちる・押し出されるような水平方向の力（波力）が自然に発生します。
